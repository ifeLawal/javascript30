# 09 Dev Tools Domination

Worked with dev tools for logging errors and checking code. 
Ones I found most exciting: 
- console.assert
- console.table
- console.group & console.groupEnd

Not as useful to me currently, but good to know:
- console.log
- console.clear
- console.error
- console.info
- console.warn
- console.count
- console.dir
- console.time
